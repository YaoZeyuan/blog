推荐线路：

  • 首日：
    • （13:30抵达）城规馆->海滨步行道->夜里->情人坝->五四广场->睡觉休息
  • 次日
    • 鲁迅公园->海滨木栈道->八大关->二浴->花石楼->中山路天主教堂(附近有肯德基解决午餐问题或者去吃王姐烧烤体验青岛人民的生(ku)活(bi))->(11：30抵达)江苏路基督教教堂->信号山公园->老舍故居->海大->百花苑->小西湖看一眼
  • 然后
    • 浮山+轮渡（市内，耗时半天）、北九水（山区，耗时一天）任选
    • 栈桥随意，下火车直接去都行

景点分类

  • 日场

    • 清晨
      • 北九水
        • 游览时间
          • 整个白天(10:00--18:00)
        • 景区简介
          • 北九水景区整体由一条瀑布引下的溪流组成，溪水凉爽清澈，河滩内有石块，可以下河玩水
          • 推荐穿着凉鞋短裤过去，毕竟在北九水就是玩水的嘛。最好带条毛巾，别不小心掉进去咯~哈哈
        • 前行方式
          • 北九水在汽车东站坐619路直达，若住在青科校内推荐在，定点发车。
            • 从汽车东站出发时间为6:00/7:00/9:00/11:00/13:00/15:00
            • 由北九水返回时间为7:40/8:40/10:40/12:40/14:40/16:40
          • 总之，11点前坐619到终点站北九水站下车，然后跟着旅游大巴走500米来到景区门口，买票或者逃票均可
          • 附图
            • 建议在辽阳东路站坐619，北九水站下车，站牌如图
            • 坐619至终点站后，沿红色箭头方向前行至停车场，左侧即为北九水入口
            • 检票口前的小桥边上有条小路，可以从小路走，绕过检票口后在从山边下山，趟过溪水进入景区
            • 建议走红色标出的那条线，爬山线没啥意思。。。还累
            • 在里面玩水那是必须的
            • 当然别忘了初中学过的折射原理
      • 崂山
        • 不建议去。如要去可搜索去『大河东客服中心』的路线，在那里下车、购票，上山即可。末班下山索道4点，从检票口到客服中心的末班大巴为5点左右

    • 上午

      • 鲁迅公园
        • 青岛的基础景点之一，碧海礁石，偶有浪涛拍岸，着凉鞋下海尤佳
      • 花石楼
        • 门票八块五，凭学生证半价。
        • 青岛有一特点，真正好玩的地方大多不收门票或只象征性的收点以控制人流，比如城规馆，比如炮台遗址，但凡漫天要价的，基本都是渣，比如太清宫、啤酒博物馆、崂山以及啤酒节。二话时候的门票价格，摆明了就是冲着控制人流去的（节假日旅游团党人表示这门票价格毫无压力= =）
        • 花石楼早先为宜富商所见，后被征用为陈毅避暑私宅，屋内有背景介绍可以一看，沿楼梯向上走有一观（liao）景(wang)台（shao），可以在此处小驻，看二浴沙滩，看海涛礁石，以及，拍照~

      • 二浴

        • 过了八大关就是第二海水浴场，沙软无浪，是市内看海的好地方
        • 然后是图片展示区

        • 沿山路爬到山顶之后可以看到青岛全貌，小屋在北面可上，北侧山头亦可爬，其上为青岛市内至高点，环览青岛，从石老人一路看到跨海大桥

      • 海边栈道
        • 青岛城市规划展览馆至极地海洋世界
          • 离开城市规划馆，沿海岸一路前行直至极地海洋世界，这是是青岛滨海步行道的起始部分，也是整个崂山区最漂亮的一条路。沿路是海滨公园，途中还会路过海滨雕塑园（不推荐进去）。趁着阴天，一路逛逛，看看海浪，非常舒服
          • 从城规馆到青岛火车站全长40.6公里，亲测一天可以走完，体力好不妨一试~

      • 江苏路基督教教堂
        • 曾在那里见过一位正信的女生，周末在教堂里义务传福音，协调游览秩序，周身有着阳光的味道。。。好吧，其实我想说，在哪里，可以见到真正的信徒，他们会让你感受到信仰的美好
        • 教堂里有钟楼，里面的机械钟在工作了近100年之后，仍然清晰洪亮。每个星期天的12点整都会鸣响30s以上，钟楼下面就是教堂。以下内容好妹子看不见：内心OS:如若在那时表白，相信会是一段很浪漫的记忆吧
        • 门票10元
        • 友情提示
          • 美国领事馆旧址就在马路对面
          • 路对面的德式建筑可以进去，里面有时还会举办画展
          • 胶澳总督府旧址距此处直线距离322米
          • 此处为欧式建筑群中心地带
          • 老舍故居（骆驼祥子博物馆，免票）距离此处直线距离478米
          • 其实我的意思是，这里是青岛景点的核心地带，随便逛逛都是景点，只是不认识罢了
          • 不过也无妨，反正都刷了那么多人生成就了，不缺这两项
          • 在青岛好好放松下呗
          • 从教堂出发200米处是信号山公园，山顶有旋转观景厅，门票20，也可一看
          • 空闲很多的话可以去老舍故居与信号山公园，空闲再多些的话可以看其他地方，正常情况下嘛，还是把这些全忽略上下个景点接着逛吧~
      • 中山路天主教教堂
        • 知名婚纱摄影地，只可远观，从不开门，囧。。。
        • 旁边是王姐烧烤总店，很小的一家铺子，主打烤鱿鱼烤五花肉，青岛人民评价很高，但是我带过去的所有人都觉得无所谓。如果让我推荐，我建议去王姐对面吃肯德基。。。那个比王姐好吃多了= =
      • 劈柴院（慎慎选）
        • 青岛小吃聚集地，吃过了这里任何一样小吃，你可以明白为什么王姐烧烤能在青岛如(shi)此(zai)有(nan)名(chi)
        • 劈柴院里的豆腐脑基本可以达到陈明非家楼底下那家的水平
        • 王姐烧烤里的鱿鱼不如核桃园那块的胡家麻辣烫
        • 爆肚各种难吃，还贵！
        • 第一次逛劈柴院的时候，回来就拉肚子了= =
        • 劈柴院，专坑外地游客的一线品牌，口亨~
      • 青岛市博物馆（慎选）
        • 位于青岛啤酒城内，下午四点前以去博物馆的名义可以直接免票进啤酒城城内
        • 其内有青岛历史文物展示
        • 优点是文物均为真品，很详实
        • 缺点是青岛本身历史较短，没啥值得看的东西。。。具体参见大洋彼岸的美国国家历史博物馆= =
    • 下午
      • 八大关
        • 在武胜关路下车即到，秋有落叶春有花，也是个拍婚纱照的胜地。所谓八大关实际上是八条以关隘命名的路名，这八条路所覆盖的区域则被称为八大关。每条路上的行道树种类都不一样，道旁有文字介绍，路过时可以一看
        • 但是不推荐冬夏来这里，到了之后随意逛逛走到二浴/花石楼就好了，夏天太热，冬天只剩下了松树还在绿着
        • 在这方面，八大关不如百花苑好，百花苑任何一个季节都可以去，目力所及尽皆美景，当然，在满街黄叶的壮丽程度和四面飘香的浪漫度上，百花苑作为一个小公园还是比不了这个街区的
        • 不过反正青岛这么大景点这么多，去哪不去哪儿，who care~
        • 图片展示区：
          • 秋日八大关

      • 湛山寺
        • 可以一去，门票十元，学生半价
        • 寺内人不多，只偶尔有一两位走过，各殿前有文字介绍，可以在寺内体验一下佛教的氛围
        • 寺外有放生池，其内金鱼成群，去的路上带点吃的，喂喂鱼倒也不错
        • 寺门处有不少野猫，带的吃的也可以分给他们一点，安卓手机再装上『猫叫模拟器』，恭维一下猫大人，有点意思~
        • 下午五点闭寺，下午的景色也很好，在那儿照点像吧~
        • 附图：湛山寺内景
      • 轮渡
        • 推荐所有没坐过船或看过海上日落的同学都来坐坐，在船舷上看大陆渐渐远去，夕阳缓缓沉入海面，再配上迎面的海风耳机里播的我心永恒，你会深深的体会到这句话的感觉
        • 我现在，就缺一个女朋友/男朋友了。嗯
        • 当然，这其实是交通工具，7:15首发船,18:15末班船，30分钟一班。坐的时候算好点，我和我姐坐过去之后就没回去的船了= =最后又穿海底隧道过来的= =

      • 栈桥
        • 栈桥就在火车站旁边，出火车站提着行李等公交车的时候就看到。
        • 单论栈桥来说，景色平淡无奇，栈桥尽头的回澜阁内还恭恭敬敬的矗立着一块水泥板号称『千古之谜无字碑』方便游客测试自己的智商。
        • 只是普通的石桥，只是普通的海。
        • 但是，很多人是在这里，才第一次见到了大海，第一次看见了礁石滩，第一次在栈桥下的礁石摊上捉到了海星螃蟹，第一次见到了清澈湛蓝的天空，美到让人流泪
        • 所以，找个下午来一趟这里吧，看一看天，看一看海，如果是冬天的话，海鸥也会在你身旁，带点面包喂喂，何乐而不为？

        • 海洋大学，又称中国每羊大学，没办法，自来水管老坏，严重缺水
        • 不过这并不影响我们亲爱的海大成为青岛市内文艺青年出产比例最高的地方，考虑到这里曾是德国人经营青岛的核心所在，所以海大拥有青岛市内最佳春景和顶尖的城市规划设计，自然也是顺利成章的事（内心os：显然只修了海大的下水道XD）
        • 春天来过几次海大，每次来到这里，都会醉在那路旁的花香，浓郁的绿茵里
        • 但是夏秋两季，倒真是很少前来，所以对其内景区质量不敢打包票
        • 不过，有空就来吧，买点海大草稿纸回去装学霸也很有意思哇
        • 更何况，还打旁边的大学路，可是青岛少数几个有卖手绘地图的地方呢~

      • 小倩倩馄饨(晚饭)
